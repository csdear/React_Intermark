# React_Intermark
React Intermark class, scotch io 

Prerequisites : 
1.  Install Node.Js
2.  Install Yarn Pkg manager
      -  Mac : brew install yarn
      -  Windows : download and install the msi
3.  Install webpack and some dependencies :   
      -  yarn add webpack webpack-dev-server path
4. Run it :  
      -  yarn start
